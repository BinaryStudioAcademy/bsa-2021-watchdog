Add Watchdog to Startup.cs as service:
Then add to appsettings.json your API key:
Now Watchdog can track all unhandled exceptions that occur when the client application is running. To track exceptions in any try-catch statements or custom exception handlers, you need to create a Watchdog client using a provider, which is passed to the class constructor with DI container and then call necessary methods.
